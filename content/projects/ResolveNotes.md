# Davinci resolve notes

This is not a project by my way of quickly editing some (drone) footage

## Stabalise

Add all videos into Gyroflow and run it against to profile for Caddx Moonlight.

## Project Setup

- Import media Folder instead of files
- set backup in Preferences -> User


## Rought cut

In cut view:

- hover over preview Clip
- set in Position with "i"
- set out Position with "o"
- use "shift+f12" to insert section at the end
- use "q" so toglle source and timeline view
- "alt+x" for remove IOs


## Detailed cut

- mouse click on the boarder between clips
- use upper windows to adjust cut (left out , right in)
- "j,k,l" (play back, stop, play)
- "/" play around cut
